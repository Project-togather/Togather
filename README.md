# 프로젝트명 To Gather

저희 프로젝트는 "취미 기반 소모임 참가" 라는 주제를 가지고 있습니다. <br>
이에 To Gather 은 두가지 의미를 가지고 있습니다. <br><br>
"함께" 의 의미인 Together. <br>
"모으다" 의 의미인 To Gather. <br><br>
자신과 취미가 비슷한 사람들을 "모으고" 취미를 "함께" 한다는 중의적인 의미를 담았습니다.



<br><br>
# 프로젝트 설명

**To gather - 다양한 취미부터 미라클 모닝같은 챌린지까지, 모두 To Gather 에서 해결 하세요**
 
<br>
단기(1회성)모임, 장기모임, 챌린지 등 다양한 유형의 모임을 즐길 수 있는 곳 입니다.

채팅, 알람, 깔끔하고 직관적인 UI 를 통해 모임을 생성하고 참가해보세요!

 
 
  <div style="display:flex; flex-direction:row;">

 - 채팅
 - 알람
 - 유료 / 무료 모임
 - 카테고리 별 검색
 - 카카오 맵 API
 - 카카오 페이 API
 
</div>
<br><br>


# 개발배경
 <div style="display:flex; flex-direction:row;">

소수 취향, 취미의 수익화 등 다양한 분야의 취미가 떠오르지만 위험한 사회 범죄 현상의 증가, 사회적 교류가 줄어드는 현 시대. 
자신의 취미를 공유하고 취향이 맞는 사람들과 함께 시간을 보내며 느끼는 행복이 필요하지 않을까 라는 생각을 시작하게 되었습니다.
대표적인 모임 기반 웹 사이트 3가지를 찾아보았을 때,
1. 카테고리 별로 내가 원하는 모임만 구분지어 검색하는 기능의 부재
2. 모임 인원의 제한이 있음에도 불구하고 알림기능이 없어서 마감으로 참가하지 못하는 상황
<br>
이 두가지의 부재를 느껴 이를 보완한 사이트를 만들기로 결정하였습니다.


</div>
